# llamdrop modules

This folder contains the core Python modules. Each module does one job.

| File | Purpose | Status |
|---|---|---|
| `device.py` | Reads RAM, CPU cores, chipset, OS, storage without asking user anything | 🔨 In development |
| `browser.py` | Interactive model browser — verified catalog + live HuggingFace search | 🔨 In development |
| `downloader.py` | Downloads models with progress bar, auto-picks quantization, resumes if interrupted | 🔨 In development |
| `launcher.py` | Wraps llama.cpp with automatically tuned flags for your device | 🔨 In development |
| `chat.py` | Chat loop with context trimming and live RAM monitor | 🔨 In development |

> All modules are coming in v0.1. Watch the repo for updates.
